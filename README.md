# BiblioHub

## Description
Ce projet est un site web de **bibliothèque** réalisé en HTML, CSS et JavaScript  
Le site permet de présenter la bibliothèque, ses horaires, son règlement, les catégories de livres disponibles, un espace livre, et une page de contact.

Chaque membre a travaillé sur une page ou une partie du projet dans sa propre branche GitHub

---

## Technologies utilisées
- HTML
- CSS
- JavaScript
- GitHub (branches, commits, pull requests)
- GitHub Pages (déploiement)

---

## Structure du projet

### Pages principales
- `index.html` : page d’accueil
- `horaire.html` : horaires d’ouverture
- `reglement.html` : règlement de la bibliothèque
- `categorie.html` : catégories de livres
- `livre.html` : page livres / liste de livres / catalogue
- `contact.html` : page contact

### Dossiers
- `/css` : contient tous les fichiers CSS
- `/images` : contient les images (logo, photos, etc.)

### CSS
- `global.css` : style général du site (commun à toutes les pages)
- un fichier CSS par page (ex : `css/contact.css`, `css/index.css`...)
